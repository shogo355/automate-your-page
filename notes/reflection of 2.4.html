2.4 Control Flow and Loops:If and While

What is a control flow statement?

A control flow statement is a statement that would make a choice. In Python, if and while are the example of control flow statement.

Comparison operators

The comparison operators can compare the two values and return if the statement is True or False. These True and False are considered as a Boolean value.
The example of comparison operators are: >,<,!=,<=,==

How do you use if statement?

An if statement can be used with comparison operators.An example was given below.

if a>b:
b=b+2
Here, it made a comparison whether a is bigger than b or not. If this statement is True, the next line will be conducted(b=b+2). On the other hand, if the statement False then the next line will NOT be conducted.

How do you use while statement?

A while statement is similar to if statement. The difference is that it loops until the while statement become False. Here is the example:

while a>b:
b=b+2
In this example, the "b=b+2" iterates until "a>b" become False.
